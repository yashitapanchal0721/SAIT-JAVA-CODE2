# SAIT-JAVA-CODE2
# ☕ Core Java

This repository contains Core Java programs that demonstrate the fundamental concepts of Java programming. It is designed for beginners who want to learn Java from basic to intermediate level through simple examples.

## 📚 Topics Covered

- Java Basics
- Variables and Data Types
- Operators
- Conditional Statements (if-else, switch)
- Loops (for, while, do-while)
- Arrays
- Strings
- Methods
- Object-Oriented Programming (OOP)
  - Classes and Objects
  - Constructors
  - Inheritance
  - Polymorphism
  - Abstraction
  - Encapsulation
- Exception Handling
- Collections Framework
- File Handling
- Multithreading
- Packages
- Input and Output (I/O)

## 🛠️ Technologies Used

- Java
- JDK 8 or above
- VS Code / IntelliJ IDEA / Eclipse

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/core-java.git
```

2. Navigate to the project folder:

```bash
cd core-java
```

3. Compile the Java file:

```bash
javac FileName.java
```

4. Run the program:

```bash
java FileName
```
